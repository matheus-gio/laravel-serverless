# Install serverless framework

``` npm install -g serverless ```

# AWS SDK Configure

``` aws configure ```


# Setup
Install packages

``` composer install ```


### Generate Assets

``` npm run watch ```


### Deploy Development
``` composer deploy-dev ```

### Deploy Production
``` composer deploy-prod```
